# archive/

**Purpose:** Documents that have been superseded but are kept for traceability.

## Convention

When a document is replaced, move it here with the path preserved:
- `core/OLD_DOC.md` → `archive/core/OLD_DOC.md`
- Add a `[ARCHIVE]` entry to CHANGELOG.md
- Update any `dryad-docs://` references that pointed to the old document
