# Dryad AI Knowledge Vault

Source of truth for architecture, planning, and research docs.

## Structure

| Folder | Purpose |
|--------|---------|
| `00_inbox/` | Unprocessed notes, scratch, quick captures |
| `10_dryad/` | Core technical knowledge |
| `10_dryad/architecture/` | System design, diagrams, decisions |
| `10_dryad/phases/` | Phase docs and specs |
| `10_dryad/nodes/` | Per-node config notes |
| `10_dryad/runbooks/` | Operational procedures |
| `10_dryad/post-mortems/` | Incident write-ups |
| `20_research/` | External reading, experiments, spikes |
| `30_planning/` | Roadmaps, strategy, business planning |
| `40_brand/` | Messaging, positioning, naming |
| `50_consulting/` | Client-facing material |
| `90_meta/` | Vault config, changelog |
| `_attachments/` | Images, PDFs, binary assets |
